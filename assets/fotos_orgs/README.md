# Fotos de Organizaciones

Coloca aquí las fotos de los proyectos ejecutados por las organizaciones.

## Formato recomendado:
- Nombres: foto1.jpg, foto2.jpg, foto3.jpg, etc.
- Formato: JPG o PNG
- Resolución recomendada: 1920x1080 o superior
- Las fotos se mostrarán de forma aleatoria como fondo de la portada

## Instrucciones:
1. Guarda las fotos con nombres secuenciales (foto1.jpg, foto2.jpg, etc.)
2. Actualiza el array `fotosOrgs` en index.html si cambias los nombres
3. Las fotos se recortan automáticamente para ajustarse al ancho completo
